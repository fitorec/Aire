Audiotoria de redes

Aire.sh es un script elaborado para facilitar la auditoria de redes inalambricas. Su fin es semi-automatizar el crackeo de WEP keys (únicamente WEP, no WPA) con una poca interveción del usuario. Fue desarrollado con fines puramente educativos ;) Lxs colaboradorxs de este proyecto no se hacen responsables por el mal uso ke se le pueda dar. Come frutas y verduras.
